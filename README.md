# MicroBlog
  Para rodar a ui deve-se ter a seguintes requisitos:
  - Angular CLI 10
  - Node acima de 8
  - Navegador web
# Iniciando o projeto
  - Rode os seguintes comandos na pasta do projeto: 
  ```
  npm install
  ```
  ```
  ng serve --open
  ```
  - Inicie a API, que encontra-se em: https://github.com/developerRafu/micro-blog-api
